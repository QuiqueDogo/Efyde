Esto es una prueba para Edyfe usando Angular y bootstrap

## Como empezar

Primero instalamos paquetes
```bash
npm i 
```
despues corremos el proyecto:

```bash
ng serve
```

Abrimos en [http://localhost:3000](http://localhost:3000) para ver la prueba.
